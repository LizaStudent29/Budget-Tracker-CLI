# Budget Tracker CLI

Минималистичный CLI‑трекер бюджета на TypeScript.

## Структура проекта
```
budget-tracker-cli/
├── node_modules/
├── src/
│   └── index.ts
├── dist/
│   └── index.js
├── package.json
├── tsconfig.json
└── README.md
```

## Установка
```
npm install
```

## Компиляция
```
npx tsc
```

## Запуск
```
node dist/index.js
```
